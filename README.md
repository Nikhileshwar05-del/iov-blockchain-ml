# Blockchain-Enabled IoV Malicious Node Detection

## Overview
Machine learning + blockchain system to detect malicious vehicles in IoV networks.

## Features
- IoV simulation with 200+ vehicles
- Trust score calculation
- Hybrid blockchain consensus
- ML models (Random Forest, SVM, Logistic Regression)

## Tech Stack
Python, Machine Learning, Blockchain

## Results
Achieved high accuracy in detecting malicious nodes.
